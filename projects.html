Description:

Develop a shopping cart system that allows users to browse products, add them to the cart, and make a purchase. The application will enable users to:

Browse a list of products available for purchase

Add products to a shopping cart

Remove items from the cart

View the total price of the cart

Proceed to checkout by entering user details like name, address, and payment information

Key Features:

MySQL database to store products, including name, price, and category

Display products dynamically from the MySQL database in the user interface

Allow users to add/remove products from the shopping cart

View total price of the items in the cart

Checkout functionality to process user orders (name, address, payment details)

Technologies Used:

MySQL: Used for storing the product catalog and order information.

Tkinter: Used for building the graphical user interface (GUI) to display the product list, manage the shopping cart, and handle the checkout process.
